Booktorious

A proof of concept EPUB reader implemented in JavaScript.

Any EPUB from epubbooks.com should work.

You can also drag & drop a file into the browser window.

EPUBs packed with the Java zip libraries, which is using a different Zip type than the one in the EPUB spec, doesn't work.

Your browser does not support the HTML5 FileReader object.

It's in the HTML5 spec, but as of now, the only known browsers implementing it are:

  • Firefox 3.6 or newer
  • Chrome 6 or newer

About the project

Booktorious is an open source 100% JavaScript webapp free of server side logic. It is utilizing new html5 technologies to read and handle files directly in the JavaScript runtime - everything happens in JavaScript.

It is using the following libraries (all except js-inflate written by me):

js-epub
Reads the EPUB file format, providing an array DOM objects of the chapters in the book. It also converts all images in the book to data uris.
js-unzip
EPUB files are packaged as Zip files.
js-inflate
Zips are containers that can use many different compression formats. EPUB books use deflate (compression) and inflate (decompression). js-unzip itself doesn't implement decompression, it justs tells ut what compression algorithm the Zip file is using.
Sarge
My own DOM library. Call me crazy.
Mediocre
The JavaScript application framework I wrote. Unlike Cappuccino and Sproutcore, it doesn't abstract away HTML and CSS. It's not finished or production ready at all.
Copperplate
JavaScript templating engine with HTML-like synax. Used by Mediocre.

All of which are are MIT licensed.

What it doesn't do

  • Work in Internet Explorer. Sarge, the DOM library I use, is awesome. Sarge itself is just an API that doesn't do any DOM work. You use plugins to implement the actual DOM manipulation functionality. The plugin that I use is home brewed, and doesn't care about IE at all. Please blame me and not sarge for this.
  • Attempt to provide a great reading experience. Booktorious is just a tech demo. It doesn't remember your position in a book, everything is in one huge scroll, and so on.

Contributing/hacking

You need Ruby and Rubygems installed to run the Mediocre framework server.

  • git clone git://github.com/augustl/booktorious.git
  • cd booktorious
  • gem install mediocre
  • mediocre server

A HTTP server is now running http://localhost:9292.
